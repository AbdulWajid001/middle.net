# React Middle
This is the React Middle application.

## Prerequisites

- [Node v16](https://nodejs.org/en/docs/)
- [npm](https://docs.npmjs.com/)

### 1- Clone Repository:
    git clone https://github.com/arbisoft/middle.git

### 2- Change Directory:
    cd frontend

### 3- Install Dependencies:
    npm install

### 4- Start Server:
    npm start
